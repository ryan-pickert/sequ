SEQU

The modular sequencing program

About SEQU

SEQU is a sequencing program that is built on top of a digital form of modular synthesis. This allows free and easy control over multiple sound sources and their parameters.

Features

  • Multiple types of synth modules and samplers
  • Multiple effect modules including reverb, delay, chorus, etc.
  • LFO modules to modulate any parameter on any module
  • MIDI input
  • Availiable on all platforms
